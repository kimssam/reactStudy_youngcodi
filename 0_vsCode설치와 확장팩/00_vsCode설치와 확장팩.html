emmet설치후 tab설정

open in broswer : alt+b로 브라우저 띄우기

auto rename tag : 시작태그와 끝태그를 한꺼번에 수정해줌

beauty : 영역 선택후 ctrl+shift+B를 클릭하여 자동 줄맞춤

css peek : 클래스명 선택하면 외부 스타일 시트의 해당 위치로 이동

live server : html파일 작성후 저장하면 브라우저에 바로 반영 =>live server로 브라우저 확인시 크롬으로 기본 브라우저 설정=> live server config :chrome, Live Server › Settings: Host 변경

html end tag labels : 끝태그에 자동으로 해당 태그의 클래스명이나 아이디가 적용되었는지 보임

live preview : html 입력 바로바로 보여지려면 설치후 우측상단 split show preview 클릭

code runner : js 입력시 콘솔창에서 바로바로 보여지려면 설치후 우측상단의 play버튼 (ctrl+alt+n) 클릭 => node.js(자바 스크립트 런타임으로 과거 웹 브라우저에서만 동작하던 js를 기타 프로그램에서도 동작 가능함)가 설치되어야 글자가 깨지지 않음

바닐라 스크립트 : 순수한 자바스크립트

모던 자바스크립트 : 2015년 es6문법을 사용하면서 리액트, 뷰, 앵귤러 등 가상 DOM을 이용하는 프레임워크를 사용하고 패키지 관리자(npm:node package manager)를 사용하면서 SPA(하나의 페이지에서 동적으로 다른 페이지를 로드해서 불러오는 페이지) 페이지인 특징을 가진 자바스크립트

타입 스크립트 : 더 구체적으로 변수의 타입을 가지는 형태의 스크립트
